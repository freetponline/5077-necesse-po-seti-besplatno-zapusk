Запуск Necesse по сети в мультиплеер:

2.1) Запускаем Стим и входим в свой акк.

2.2) Запускаем игру файлом \Necesse.exe

  3.1) Многопользовательская Игра => Создаём Новый Мир / Указываем ранее созданный => Указываем остальные параметры и попадаем в игру => Видимость сервера рекомендую указать ОТКРЫТЫЙ => После того как попали на сервер можем пригласить друзей (ESC => Пригласить Друзей => Выбираем друзей)
